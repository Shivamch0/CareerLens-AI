🚀 CareerLens AI (Frontend - Initial Phase)

CareerLens AI is an AI-assisted career guidance platform designed to help users discover suitable career paths based on their interests, skills, and academic background.

«⚠️ Current Status: Frontend UI completed (Login, Signup, Home, Dashboard) using dummy data.»

---

📌 Features Implemented

🔐 Authentication Pages

- Login Page
  
  - Clean UI design
  - Input validation (basic)
  - Styled with modern UI components

- Signup Page
  
  - User registration form
  - Fields for user details (username, email, password)
  - Responsive layout

---

🏠 Home Page

- Landing page for users
- Introduction to CareerLens AI
- Clean and minimal UI design
- Navigation to authentication pages

---

📊 Dashboard Page

- Designed UI for user insights (dummy data)
- Components include:
  - Stats Cards (skills, progress, etc.)
  - Suggestions Section
  - Visual layout for future analytics

---

🛠️ Tech Stack

Frontend

- React.js
- Tailwind CSS

⚙️ Installation & Setup

git clone https://github.com/your-username/careerlens-ai.git

# Navigate to project
cd careerlens-ai/frontend

# Install dependencies
npm install

# Start development server
npm run dev
